lcthw means "Learn C The Hard Way

How to use template:
use cpr template ex[num] and v ex[num].c to setup C environment
use :%s/1/[num]/g in vim to setup Makefile
use make/make modi/make clean to generate/clean exeutable files
use ./ex[um] argv or ./modi_ex[num] to run
use vg(valgrind) ./ex[num] to monitor execute flow

How to use c-skeleton:
just make it. test will start automatically.
